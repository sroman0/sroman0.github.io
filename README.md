# simoneromano.github.io

Personal portfolio for Simone Romano: Linux security research, eBPF exploit development, runtime observability, and digital forensics.

The site is deliberately dependency-free. It uses semantic HTML, custom CSS, and a small amount of vanilla JavaScript, and can be served directly by GitHub Pages.

## Local preview

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Structure

- `index.html` - portfolio homepage
- `case-studies/` - long-form project write-ups
- `styles.css` - design system and responsive layouts
- `script.js` - navigation and reveal behavior
- `.github/workflows/pages.yml` - GitHub Pages deployment
